Adaptive Consensus ADMM for Distributed Optimization.
Zheng Xu, Gavin Taylor, Hao Li,  Mario Figueiredo, Xiaoming Yuan, and Tom Goldstein. 
International Conference on Machine Learning (ICML), 2017.

v1, March, 2018

The package includes part of implementation in the paper: the core ACADMM solver and application on linear regression with elastic net regularization. The distributed communication is based on mpi4py. 

See the demo files for details.



The released code can be used for research purpose. Please kindly cite our paper if you find the code helpful:
